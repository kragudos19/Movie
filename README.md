## Dependencies
You can also look in the Gemfile and package.json

### Faraday Install
```exec
gem install faraday
```

### Bundler Install
```exec
    * gem install bundler 
    * https://bundler.io/
```
### Yarn Install 
```exec
    * npm install --global yarn (opt: --global)
    * https://classic.yarnpkg.com/en/docs/install/#windows-stable
```
